# InsurAI Backend

This Space hosts the backend server for the InsurAI application. It uses Flask and Gunicorn to serve the API.

The server handles:
- Media uploads (images and videos)
- AI-powered object detection and segmentation
- Item inventory management
- Claim processing logic
